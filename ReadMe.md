#　環境設定
まず python が pc に存在するかを以下のコードで確認してください.
```
py -V
```
か, 
```
py --version
```
存在しなければ公式サイトからダウンロードして下さい.

次に pip が存在するかを確認してください.

```
pip -V
```
か, 
```
pip -V
```
